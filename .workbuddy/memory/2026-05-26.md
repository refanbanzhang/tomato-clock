### 番茄时钟 Next.js 网页版迁移

将 Swift/macOS 版番茄时钟完整迁移为 Next.js 网页版。

**技术栈：** Next.js 15 + TypeScript + Tailwind CSS v4

**已实现功能：**
- 25分钟专注 / 5分钟短休息 / 15分钟长休息
- 每4个番茄触发长休息
- 定时器状态机：idle → focusing → paused → resting
- SVG 圆环倒计时进度圈
- Web Notifications API 完成通知 + Web Audio API 蜂鸣提示
- 空格键快捷键（开始/暂停/继续）
- 周进度统计（完成数/百分比/今日/上周同期/7天柱状图）
- 两种统计模式：每周一重置 / 滚动7天
- 可配置周目标（1-999）
- localStorage 持久化存储
- 完成弹窗（开始休息 / 稍后）
- 提前结束/放弃均记录为 incomplete
