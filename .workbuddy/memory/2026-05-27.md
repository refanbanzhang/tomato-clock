## 2026-05-27

### 多语言支持 (i18n)
- 新增支持 5 种语言：简体中文 (zh-CN)、繁体中文 (zh-TW)、英文 (en)、日文 (ja)、韩文 (ko)
- 实现了轻量级客户端 i18n 方案：
  - `lib/i18n/types.ts` — 翻译 key 和类型定义
  - `lib/i18n/context.tsx` — LocaleProvider + useLocale hook + `t()` 翻译函数
  - `lib/i18n/locales/*.json` — 5 种语言的翻译 JSON 文件
- 语言检测优先级：URL 参数 `?lang=xx` → localStorage → 浏览器语言 → 默认 zh-CN
- 持久化到 localStorage (`tomato-clock-locale`)
- `formatDate()` / `formatTime()` 辅助函数，自动根据 locale 选择正确的 Intl BCP47 格式
- 新增 `LanguageSwitcher` 组件（带国旗 emoji 和下拉菜单）
- 更新了所有组件：page.tsx, calendar/page.tsx, landing/page.tsx, TimerDisplay, TimerControls, SettingsPanel, AppNav, StatsSummary, CalendarView, DayDetailPanel, hooks.ts, stats.ts
- 构建通过，所有页面无报错
